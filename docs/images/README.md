# Screenshots and image assets

Drop UI screenshots here so the top-level `README.md` can render them.

Suggested filenames (already referenced from the main README):

| File                          | What it should show                                                    |
| ----------------------------- | ---------------------------------------------------------------------- |
| `project-list.png`            | The project list view with matching-rate badges.                       |
| `project-detail.png`          | The project detail view with requirement-by-requirement match list.    |
| `application-view.png`        | The generated application draft (✅/❎ bullet list).                   |
| `settings.png`                | The settings view (thresholds, polling interval, pause toggles).       |
| `architecture.png` (optional) | Architecture / data-flow diagram (can be exported from a tool of your choice). |

PNG or JPG, roughly 1200–1600 px wide is a good balance between clarity and file size.
